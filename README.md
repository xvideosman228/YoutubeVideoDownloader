# Introduction
This downloader was made with PyQT6 for interface and yt-dlp as a core.

It has 4 modes: download single video, download video queue, download all video from playlist and from channel (yet in 720p)

# Installing
Clone this repo (`git clone https://github.com/xvideosman228/YoutubeVideoDownloader`)

Create virtual environment (`python3 -m venv .venv && source .venv/bin/activate`)

Install all dependencies (`pip install -r requirements.txt`)

And run (`python3 main.py`)

*OR* make this 
```
git clone https://github.com/xvideosman228/YoutubeVideoDownloader && cd YoutubeVideoDownloader && python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt
```
